Description

Transit notification fetches real-time bus arrival data from San Francisco Bay Area 511.org and displays upcoming arrivals for a specific stop. The data is stored in a database using SQLAlchemy and shown via flask app. The website uses boostrap to implement UI experience.

Documentation

Documentation is located at https://transit-notification.readthedocs.io/en/latest/

GitHub Repo

GitHub is located at https://github.com/robertghennessy/transit_notification

What happens:

  • If the operator table is outdated or empty, query operators from 511.org and store in the operator table.
  • If the line table is outdated or empty, query lines for the selected operator from 511.org and store in the line table.
  • If the stop table is outdated or empty, query stops for the selected operator from 511.org and store in the stop table. Stop table provide information on stop name and location.
  • If the stop monitoring table is outdated or empty, query stop monitoring for the selected operator from 511.org and store in the vehicle and onward call table. Vehicle table contains information for a vehicle and onward call contains information about a vehicle reaching a stop.
